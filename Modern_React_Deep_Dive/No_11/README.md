## Next.js에서의 리액트 서버 컴포넌트

- 새로운 fetch 도입과 getServerSideProps, getStaticProps, getIntialProps의 삭제
- 정적 렌더링과 동적 렌더링
- 캐시와 mutating, 그리고 revalidating
- 스트리밍을 활용한 점진적 페이지 불러오기
